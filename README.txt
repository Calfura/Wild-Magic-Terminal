=====Wild Magic: Sorcerer - Terminal Application Program=====

===Requirements===

Colored (Python Coloring Style Module, Dimitris Zlatanidis)
D20 Module (D&D Beyond Dice Rolling Module)

===Installing===

1. Create a new folder (name of your choice), to contain all the contents of the program and needed files required to run the program.
2. Use git to create a .git directory to set all appropriate functions
3. Navigate to https://github.com/Calfura/Wild-Magic-Terminal and download the files for the program
4. Once downloaded and place in the directory of your choice, run the 'install.sh' script inside the terminal to install the virtual environment and needed modules
5. Launch program with 'run.sh'
